## 🎨 二、组件使用详情

### 组件 1：内容类型选择

**使用组件：** @component-radio-group

**布局结构：**
```
┌─────────────────────────────────────────┐
│  内容类型 *                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐│
│  │ 📝 文章  │  │ 🚀 项目  │  │ 💰 案例  ││
│  └─────────┘  └─────────┘  └─────────┘│
│  ┌─────────┐                           │
│  │ 💡 心得  │                           │
│  └─────────┘                           │
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 类型卡片 | 点击 | 选中（紫色边框） | 展开对应字段 |
| 类型卡片 | 悬停 | 上移 2px + 阴影加深 | 无 |

---

### 组件 2：富文本编辑器

**使用组件：** 前端框架自带富文本（TinyMCE/Quill）

**布局结构：**
```
┌─────────────────────────────────────────┐
│  正文内容 *                             │
│  ─────────────────────────────────────  │
│  [B] [I] [U] [H1] [列表] [引用] [图片] │
│  ─────────────────────────────────────  │
│  # 标题                                 │
│  正文内容...                            │
│                                         │
│  ─────────────────────────────────────  │
│  已输入 0 字 / 最少 200 字               │
└─────────────────────────────────────────┘
```

**功能特性：**
- 加粗、斜体、下划线
- 标题（H1-H3）
- 列表（有序/无序）
- 引用、代码块
- 插入图片、链接
- 字数统计

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 内容类型 | 类型卡片 | 点击 | 选中 | 展开对应字段 |
| 标题 | 输入框 | 输入 | 实时验证 | 显示字符计数 |

| 正文 | 富文本 | 输入 | 实时更新 | 字数统计 |
| 标签 | 输入框 | 回车 | 添加标签 | 清空输入框 |
| 提交审核 | 按钮 | 点击 | 验证表单 | POST /api/publishes |
| 取消 | 按钮 | 点击 | 弹出确认框 | 返回上一页 |

## 🔌 四、API 接口

### 创建发布接口

**请求：**
```http
POST /api/publishes
Content-Type: application/json

{
  "type": "case",
  "title": "我的副业案例分享",
  "cover_images": ["/covers/1.jpg", "/covers/2.jpg"],
  "content": "<h2>案例简介</h2>...",
  "tags": ["AI 副业", "变现", "案例"],
  "extra_fields": {
    "startup_cost": 0,
    "monthly_income": 5000,
    "time_cost": "每天 2 小时"
  }
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "title": "我的副业案例分享",
    "status": "pending",
    "created_at": "2026-04-03 10:00:00"
  },
  "message": "发布成功，等待审核"
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 内容类型：4 列布局
- 封面图：3 列布局
- 富文本：完整工具栏

平板端（768px - 1023px）：
- 内容类型：2 列布局
- 封面图：3 列布局
- 富文本：简化工具栏

手机端（< 768px）：
- 内容类型：单列布局
- 封面图：单列布局
- 富文本：最小工具栏
```
