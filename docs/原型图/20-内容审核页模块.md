## 🎨 二、组件使用详情

### 组件 1：审核列表

**使用组件：** @component-audit-list

**布局结构：**
```
┌─────────────────────────────────────────┐
│  ☐ [封面]  《我的副业案例》             │
│  ─────────────────────────────────────  │
│  📝 副业案例 | 👤 张三 | ⏰ 10:00      │
│  ─────────────────────────────────────  │
│  摘要：本文介绍了我的副业变现经验...    │
│  ─────────────────────────────────────  │
│          [✅ 通过] [❌ 拒绝] [👁️ 查看]  │
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 复选框 | 点击 | 选中/取消 | 无 |
| 封面图 | 点击 | 无 | 跳转内容详情页 |
| 标题 | 点击 | 无 | 跳转内容详情页 |
| 通过 | 点击 | 弹出确认框 | POST /api/audits/{id}/approve |
| 拒绝 | 点击 | 弹出拒绝原因输入框 | POST /api/audits/{id}/reject |
| 查看 | 点击 | 无 | 跳转内容详情页 |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新后台首页 |
| 筛选器 | 类型下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 状态下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 搜索框 | 输入 | 实时建议 | GET /api/audits/suggest |
| 审核列表 | 复选框 | 点击 | 选中/取消 | 无 |
| 审核列表 | 卡片 | 悬停 | 上移 4px | 无 |
| 审核列表 | 通过 | 点击 | 弹出确认框 | POST /api/audits/{id}/approve |
| 审核列表 | 拒绝 | 点击 | 弹出输入框 | POST /api/audits/{id}/reject |
| 批量审核 | 按钮 | 点击 | 弹出批量操作框 | 无 |

## 🔌 四、API 接口

### 审核列表接口

**请求：**
```http
GET /api/admin/audits?page=1&per_page=10&type=all&status=pending
```

**响应：**
```json
{
  "success": true,
  "data": {
    "audits": [
      {
        "id": 1,
        "publish": {
          "id": 1,
          "title": "我的副业案例分享",
          "cover_image": "/covers/1.jpg",
          "type": "case",
          "excerpt": "本文介绍了我的副业变现经验..."
        },
        "user": {
          "id": 1,
          "name": "张三"
        },
        "status": "pending",
        "created_at": "2026-04-03 10:00:00"
      }
    ],
    "pagination": {
      "total": 5,
      "per_page": 10,
      "current_page": 1,
      "last_page": 1
    }
  }
}
```

### 审核通过接口

**请求：**
```http
POST /api/admin/audits/{id}/approve
Content-Type: application/json

{
  "note": "内容优质，通过审核"
}
```

**响应：**
```json
{
  "success": true,
  "message": "审核通过"
}
```

### 审核拒绝接口

**请求：**
```http
POST /api/admin/audits/{id}/reject
Content-Type: application/json

{
  "reject_reason": "内容过于简单",
  "suggest": "建议补充更多细节"
}
```

**响应：**
```json
{
  "success": true,
  "message": "审核拒绝"
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 审核列表：完整显示
- 筛选器：横向布局
- 批量操作：完整显示

平板端（768px - 1023px）：
- 审核列表：简化显示
- 筛选器：换行显示
- 批量操作：简化显示

手机端（< 768px）：
- 审核列表：单列显示
- 筛选器：折叠显示
- 批量操作：隐藏
```
