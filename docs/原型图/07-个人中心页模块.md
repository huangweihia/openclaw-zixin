## 🎨 二、组件使用详情

### 组件 1：用户信息卡片

**使用组件：** @component-avatar + @component-badge

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [🦀 头像]  张三                  [编辑]│
│  @zhangsan  |  👑 VIP 会员  |  剩余 285 天│
└─────────────────────────────────────────┘
```

**组件属性：**
- 头像尺寸：80x80px
- 头像形状：圆形
- VIP 标识：紫色渐变
- 编辑按钮：右上角

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 头像 | 点击 | 无 | 打开头像上传 |
| 编辑 | 点击 | 无 | 打开编辑资料弹窗 |
| VIP 标识 | 点击 | 无 | 跳转 VIP 页 |

---

## 🔌 五、API 接口

### 用户信息接口

**请求：**
```http
GET /api/user/profile
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "name": "张三",
    "email": "user@example.com",
    "avatar": "/avatars/1.png",
    "vip_level": "vip",
    "vip_expires_at": "2027-04-03",
    "points": 1234
  }
}
```

### 统计数据接口

**请求：**
```http
GET /api/user/stats
```

**响应：**
```json
{
  "success": true,
  "data": {
    "publish_count": 12,
    "view_count": 1234,
    "favorite_count": 56,
    "comment_count": 23
  }
}
```

### 最近动态接口

**请求：**
```http
GET /api/user/activities?limit=10
```

**响应：**
```json
{
  "success": true,
  "data": [
    {
      "type": "publish",
      "created_at": "2026-04-03 10:00:00",
      "content": "发布了《我的副业案例》",
      "link": "/dashboard/publishes/1"
    },
    {
      "type": "favorite",
      "created_at": "2026-04-02 15:00:00",
      "content": "收藏了《AI 工具变现地图》",
      "link": "/favorites"
    }
  ]
}
```

## 📱 六、响应式规则

```

平板端（768px - 1023px）：
- 统计卡片：2 列布局
- 快捷入口：2 列布局
- 时间线：完整显示

手机端（< 768px）：
- 统计卡片：2 列布局
- 快捷入口：单列布局
- 时间线：简化显示
```
