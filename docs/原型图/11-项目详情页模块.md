## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 项目信息 | 收藏 | 点击 | 变色 | POST /api/projects/{id}/favorite |
| 项目信息 | 分享 | 点击 | 弹出菜单 | 无 |
| 正文 | 内部链接 | 点击 | 无 | 跳转对应页面 |
| 评论 | 点赞 | 点击 | 数字 +1 | POST /api/comments/{id}/like |
| 评论 | 回复 | 点击 | 展开回复框 | 无 |

## 🔌 四、API 接口

### 项目详情接口

**请求：**
```http
GET /api/projects/{id}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "title": "AI 写作助手",
    "cover_image": "/covers/1.jpg",
    "content": "## 项目简介...",
    "monthly_income": "5000+",
    "startup_cost": 0,
    "difficulty": 2,
    "tags": ["AI 工具", "写作", "副业"],
    "tech_stack": ["Claude", "ChatGPT", "Notion"],
    "is_favorite": false,
    "created_at": "2026-04-03 10:00:00"
  }
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 正文：单列，最大宽度 800px
- 变现分析：3 列布局
- 评论：完整显示

平板端（768px - 1023px）：
- 正文：单列
- 变现分析：3 列布局
- 评论：完整显示

手机端（< 768px）：
- 正文：单列，字体放大
- 变现分析：单列布局
- 评论：简化显示
```
