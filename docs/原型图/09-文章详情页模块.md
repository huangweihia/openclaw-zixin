## 🎨 二、组件使用详情

### 组件 1：文章信息区

**使用组件：** @component-badge

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [封面图]                               │
│  《AI 副业变现的 5 个方向》              │
│  ─────────────────────────────────────  │
│  💰 副业案例 | 📅 2026-04-03 | 👁️ 1,234│
│  ─────────────────────────────────────  │
│  [AI 副业] [变现] [案例]  ⭐ 收藏  📤 分享│
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 收藏 | 点击 | 变色（已收藏） | POST /api/articles/{id}/favorite |
| 分享 | 点击 | 弹出分享菜单 | 无 |
| 标签 | 点击 | 无 | 跳转标签筛选 |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 文章信息 | 收藏 | 点击 | 变色 | POST /api/articles/{id}/favorite |
| 文章信息 | 分享 | 点击 | 弹出菜单 | 无 |
| 正文 | 内部链接 | 点击 | 无 | 跳转对应页面 |
| 评论 | 点赞 | 点击 | 数字 +1 | POST /api/comments/{id}/like |
| 评论 | 回复 | 点击 | 展开回复框 | 无 |
| 发表评论 | 提交 | 点击 | 验证并提交 | POST /api/articles/{id}/comments |

## 🔌 四、API 接口

### 文章详情接口

**请求：**
```http
GET /api/articles/{id}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "title": "AI 副业变现的 5 个方向",
    "cover_image": "/covers/1.jpg",
    "content": "<h2>方向一：AI 内容创作服务</h2>...",
    "category": "case",
    "tags": ["AI 副业", "变现", "案例"],
    "view_count": 1234,
    "like_count": 56,
    "comment_count": 23,
    "is_favorite": false,
    "is_vip": false,
    "created_at": "2026-04-03 10:00:00"
  }
}
```

### 评论列表接口

**请求：**
```http
GET /api/articles/{id}/comments?page=1
```

**响应：**
```json
{
  "success": true,
  "data": {
    "comments": [
      {
        "id": 1,
        "user": {
          "name": "张三",
          "avatar": "/avatars/1.png",
          "vip_level": "vip"
        },
        "content": "这篇文章很有用，感谢分享！",
        "like_count": 12,
        "created_at": "2026-04-03 10:30:00"
      }
    ]
  }
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 正文：单列，最大宽度 800px
- 评论：完整显示
- 侧边栏：显示相关文章

平板端（768px - 1023px）：
- 正文：单列
- 评论：完整显示
- 侧边栏：隐藏

手机端（< 768px）：
- 正文：单列，字体放大
- 评论：简化显示
- 侧边栏：隐藏
```
