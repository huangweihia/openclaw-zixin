## 🎨 二、组件使用详情

### 组件 1：收藏卡片

**使用组件：** @component-card + @component-badge

**布局结构：**
```
┌───────────────────┐
│  [封面图]         │
│  AI 写作助手       │
│  🚀 项目          │
│  ─────────────────│
│  2026-04-03 收藏   │
│  ─────────────────│
│  [AI 工具] [写作] │
│  ─────────────────│
│        [🗑️ 删除]  │
└───────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 收藏卡片 | 悬停 | 上移 4px + 阴影加深 | 无 |
| 封面图 | 点击 | 无 | 跳转对应内容页 |
| 标题 | 点击 | 无 | 跳转对应内容页 |
| 删除 | 点击 | 弹出确认框 | DELETE /api/favorites/{id} |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 筛选器 | 分类下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 搜索框 | 输入 | 实时建议 | GET /api/favorites/suggest |
| 收藏卡片 | 卡片 | 悬停 | 上移 4px | 无 |
| 收藏卡片 | 标题 | 点击 | 无 | 跳转对应内容页 |
| 收藏卡片 | 删除 | 点击 | 弹出确认框 | DELETE /api/favorites/{id} |
| 批量管理 | 按钮 | 点击 | 进入批量模式 | 显示复选框 |
| 分页器 | 页码 | 点击 | 高亮当前页 | 加载对应页数据 |

## 🔌 四、API 接口

### 收藏列表接口

**请求：**
```http
GET /api/favorites?page=1&per_page=9&category=all
```

**响应：**
```json
{
  "success": true,
  "data": {
    "favorites": [
      {
        "id": 1,
        "favoritable_type": "App\\Models\\Project",
        "favoritable_id": 1,
        "title": "AI 写作助手",
        "cover_image": "/covers/1.jpg",
        "category": "project",
        "tags": ["AI 工具", "写作"],
        "created_at": "2026-04-03 10:00:00"
      }
    ],
    "pagination": {
      "total": 56,
      "per_page": 9,
      "current_page": 1,
      "last_page": 7
    }
  }
}
```

### 批量删除接口

**请求：**
```http
DELETE /api/favorites/batch
Content-Type: application/json

{
  "ids": [1, 2, 3]
}
```

**响应：**
```json
{
  "success": true,
  "message": "已删除 3 个收藏"
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 收藏卡片：3 列布局
- 筛选器：横向布局
- 分页器：完整显示

平板端（768px - 1023px）：
- 收藏卡片：2 列布局
- 筛选器：换行显示
- 分页器：简化显示

手机端（< 768px）：
- 收藏卡片：单列布局
- 筛选器：折叠显示
- 分页器：仅显示上一页/下一页
```
