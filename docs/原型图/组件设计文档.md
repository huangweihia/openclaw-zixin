> **版本：** v4.1

## 📝 零、顶部导航栏详细设计（⭐核心组件）

### 组件 0：顶部导航栏（@component-top-nav）⭐核心组件

**使用位置：** 所有页面顶部

**设计依据：**
- 功能清单：00-公共功能模块（6 个导航功能）
- 数据库字段：users.vip_level, users.unread_notifications_count, users.avatar, users.name, users.points

**组件结构：**
```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  🤖 AI OpenClaw 智信  |  首页  文章  项目  案例  工具  |  [🔍]  [🔔]  [👤▼]     │
│                     │                               │      ┌──────────────┐   │
│                     │                               │      │ 个人中心     │   │
│                     │                               │      │ 我的收藏     │   │
│                     │                               │      │ 我的发布     │   │
│                     │                               │      │ ──────────── │   │
│                     │                               │      │ 👑 VIP 会员   │   │
│                     │                               │      │ 剩余 285 天    │   │
│                     │                               │      │ ──────────── │   │
│                     │                               │      │ 设置         │   │
│                     │                               │      │ 退出登录     │   │
│                     │                               │      └──────────────┘   │
└─────────────────────────────────────────────────────────────────────────────────┘
```

**组件属性：**
- 高度：64px
- 背景：白色，带阴影
- 位置：fixed w-full z-50
- Logo 区：200px
- 菜单区：自动
- 用户区：200px

**功能特性（结合数据库字段）：**

| 区域 | 元素 | 数据绑定 | 显示逻辑 | 交互 |
|------|------|---------|---------|------|
| Logo | Logo 图标+文字 | - | 始终显示 | 点击刷新首页 |
| 主菜单 | 首页/文章/项目/案例/工具 | - | 始终显示 | 点击跳转对应页面 |
| 搜索框 | 搜索图标 | - | 始终显示 | 点击展开搜索框 |
| 通知 | 通知图标 + 红点 | `users.unread_notifications_count` | >0 显示红点 | 点击跳转通知中心 |
| 用户头像 | 头像/昵称首字母 | `users.avatar`, `users.name` | 已登录显示 | 点击展开下拉菜单 |
| VIP 标识 | 👑 图标 | `users.vip_level` | vip/svip 显示 | 点击跳转 VIP 页 |
| 积分 | 积分余额 | `users.points` | 已登录显示 | 纯展示 |
| 下拉菜单 | 个人中心/收藏/发布/设置/退出 | - | 已登录显示 | 点击跳转对应页面 |

**用户状态显示逻辑：**

| 用户状态 | 显示内容 |
|---------|---------|
| **未登录** | Logo + 菜单 + 搜索 + [登录] [注册] |
| **免费用户** | Logo + 菜单 + 搜索 + 通知 + 头像下拉 |
| **VIP 用户** | Logo + 菜单 + 搜索 + 通知 + 头像+VIP 标识 + 下拉 |
| **SVIP 用户** | Logo + 菜单 + 搜索 + 通知 + 头像+SVIP 标识 + 下拉 |

**下拉菜单结构：**

| 元素 | 点击/悬停 | 反应 | 动作 |

## 📝 二、组件详细设计

### 组件 1：标签输入框（@component-tag-input）

**使用位置：** 发布页标签输入、SVIP 订阅关键词

**组件结构：**
```
┌────────────────────────────────────────────────┐
│  标签（3-5 个）                                 │
│  ┌──────────────────────────────────────────┐ │
│  │ AI 副业  [×]  变现  [×]  案例  [×]       │ │
│  │ _                                        │ │
│  └──────────────────────────────────────────┘ │
│  推荐：AI 工具  运营 SOP  资源分享            │
└────────────────────────────────────────────────┘
```

**组件属性：**
- 宽度：100%
- 最小高度：48px
- 边框：1px solid #e5e7eb
- 圆角：8px
- 标签背景：#f3f4f6

**功能特性：**
| 功能 | 说明 |
|------|------|
| 输入标签 | 输入后按回车添加 |
| 删除标签 | 点击标签上的 × 删除 |
| 推荐标签 | 点击推荐标签快速添加 |
| 数量限制 | 最少 3 个，最多 5 个 |
| 重复检测 | 不能添加重复标签 |
| 长度限制 | 单标签 2-10 字符 |

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 输入框 | 输入 | 显示输入内容 | 无 |
| 输入框 | 回车 | 添加标签 | 清空输入框 |
| 标签 × | 点击 | 删除标签 | 无 |
| 推荐标签 | 点击 | 添加到已选标签 | 无 |
| 推荐标签 | 悬停 | 背景变色 | 无 |

**Props 配置：**
```php
[
    'model' => 'tags',              // v-model 绑定字段
    'placeholder' => '输入标签后回车', // 占位文字
    'minTags' => 3,                 // 最少标签数
    'maxTags' => 5,                 // 最多标签数
    'minLength' => 2,               // 单标签最小长度
    'maxLength' => 10,              // 单标签最大长度
    'suggestions' => [],            // 推荐标签列表
]
```

**Blade 使用示例：**
```blade
<x-tag-input 
    model="tags"
    placeholder="输入标签后回车，3-5 个"
    :min-tags="3"
    :max-tags="5"
    :suggestions="['AI 副业', '变现案例', '运营 SOP', '工具推荐', '资源分享']"
/>
```

### 组件 3：文件上传组件（@component-file-upload）

**使用位置：** 个人中心头像上传、发布页附件上传

**组件结构：**
```
┌────────────────────────────────────────────────┐
│  头像上传                                      │
│  ┌──────────────────────────────────────────┐ │
│  │          ┌─────────┐                     │ │
│  │          │  👤     │                     │ │
│  │          │ 头像     │                     │ │
│  │          └─────────┘                     │ │
│  │                                          │ │
│  │  点击或拖拽文件到此处上传                 │ │
│  │  支持 JPG/PNG/GIF，最大 2MB               │ │
│  └──────────────────────────────────────────┘ │
└────────────────────────────────────────────────┘
```

**组件属性：**
- 宽度：100%
- 高度：200px（头像）/ 120px（附件）
- 边框：2px dashed #e5e7eb
- 圆角：8px
- 拖拽悬停：边框变紫色

**功能特性：**
| 功能 | 说明 |
|------|------|
| 点击上传 | 打开文件选择器 |
| 拖拽上传 | 拖拽文件到上传区 |
| 格式验证 | 验证文件类型 |
| 大小验证 | 验证文件大小 |
| 预览 | 上传后显示预览图 |
| 进度条 | 显示上传进度 |
| 删除 | 删除已上传文件 |

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 上传区 | 点击 | 打开文件选择器 | 无 |
| 上传区 | 拖拽文件进入 | 边框变紫色 + 背景变色 | 无 |
| 文件选择 | 选择文件 | 显示文件名 + 大小 | 开始上传 |
| 上传中 | - | 显示进度条 | 无 |
| 上传成功 | - | 显示预览图 + 删除按钮 | 无 |
| 上传失败 | - | 显示错误信息 + 重试按钮 | 无 |
| 删除按钮 | 点击 | 弹出确认框 | 删除文件 |

**Props 配置：**
```php
[
    'model' => 'avatar',            // v-model 绑定字段
    'accept' => 'image/*',          // 接受的文件类型
    'maxSize' => 2,                 // 最大文件大小（MB）
    'multiple' => false,            // 是否支持多文件
    'preview' => true,              // 是否显示预览
    'uploadUrl' => '/api/upload',   // 上传接口
]
```

**Blade 使用示例：**
```blade
<x-file-upload 
    model="avatar"
    accept="image/jpeg,image/png,image/gif"
    :max-size="2"
    :preview="true"
    upload-url="/api/profile/upload-avatar"
/>
```

### 组件 7-11：列表组件（@component-*-list）

由于列表组件结构相似，统一设计基础列表组件，通过 props 配置不同场景。

**基础列表组件结构：**
```
┌────────────────────────────────────────────────┐
│  [筛选 Tab]  [筛选 Tab]  [筛选 Tab]            │
├────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────┐ │
│  │ [封面] 标题                               │ │
│  │       摘要...                             │ │
│  │       [标签] [标签]                       │ │
│  │                            阅读 123 | 赞 56│ │
│  └──────────────────────────────────────────┘ │
│  ┌──────────────────────────────────────────┐ │
│  │ ...                                      │ │
│  └──────────────────────────────────────────┘ │
├────────────────────────────────────────────────┤
│  [分页器]                                      │
└────────────────────────────────────────────────┘
```

**Props 配置：**
```php
[
    'type' => 'notification',       // notification/history/comment/publish/favorite
    'items' => [],                  // 数据列表
    'filterable' => true,           // 是否支持筛选
    'paginatable' => true,          // 是否支持分页
    'emptyText' => '暂无数据',       // 空状态文字
]
```

**Blade 使用示例：**
