## 🔌 七、API 接口

### 创建订单接口

**请求：**
```http
POST /api/payments/create
Content-Type: application/json

{
  "plan": "vip",
  "payment_method": "wechat"
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "order_no": "20260403100001",
    "amount": 99.00,
    "payment_url": "https://wx.tenpay.com/...",
    "qr_code": "data:image/png;base64,..."
  },
  "message": "订单创建成功"
}
```

### 查询订单状态接口

**请求：**
```http
GET /api/payments/order/{order_no}/status
```

**响应：**
```json
{
  "success": true,
  "data": {
    "order_no": "20260403100001",
    "status": "paid",
    "paid_at": "2026-04-03 10:05:00",
    "vip_expires_at": "2027-04-03 10:05:00"
  }
}
```

## 📱 八、响应式规则

### 断点定义

```
sm: 640px   （手机横屏）
md: 768px   （平板）
lg: 1024px  （桌面）
xl: 1280px  （大桌面）
2xl: 1536px （超大桌面）
```

### 响应式规则

```
桌面端（>= 1024px）：
- 订单卡片宽度：600px
- 支付方式：横向布局
- 步骤条：完整显示

平板端（768px - 1023px）：
- 订单卡片宽度：100% - 40px
- 支付方式：横向布局
- 步骤条：简化显示

手机端（< 768px）：
- 订单卡片宽度：100% - 32px
- 支付方式：纵向布局
- 步骤条：仅显示文字
```

## 🛡️ 九、安全说明

### 支付安全

1. **HTTPS 加密** - 所有支付请求使用 HTTPS 加密
2. **订单签名** - 订单数据签名防篡改
3. **支付回调验证** - 验证支付回调签名
4. **幂等性** - 支付回调支持幂等处理

### 资金安全

1. **微信支付官方合作** - 资金直接进入商户账户
2. **7 天无理由退款** - 支持退款保障
3. **订单记录** - 完整订单记录可追溯
