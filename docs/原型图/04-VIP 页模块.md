## 🎨 二、组件使用详情

### 组件 1：Hero Section

**使用组件：** @component-button

**布局结构：**
```
┌─────────────────────────────────────────┐
│          开通 VIP，解锁完整内容库        │
│          （48px / Bold / 渐变文字）      │
│                                         │
│   50+ 副业案例 · 20+AI 工具变现地图      │
│   · 10+ 运营 SOP                        │
│                                         │
│   [免费试用 7 天]  [查看价格方案]       │
└─────────────────────────────────────────┘
```

**组件属性：**
- 背景：渐变 #667eea → #764ba2
- 高度：400px
- 文字：白色
- 按钮：主按钮 + 次按钮

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 免费试用 7 天 | 点击 | 无 | 跳转注册页（?trial=7） |
| 查看价格方案 | 点击 | 无 | 滚动到价格区 |
| 免费试用 | 悬停 | 上移 2px + 阴影加深 | 无 |

## 📱 八、响应式规则

### 断点定义

```
sm: 640px   （手机横屏）
md: 768px   （平板）
lg: 1024px  （桌面）
xl: 1280px  （大桌面）
2xl: 1536px （超大桌面）
```

### 响应式规则

```
桌面端（>= 1024px）：
- 权益卡片：3 列布局
- 预览卡片：2 列布局
- 评价卡片：3 列布局

平板端（768px - 1023px）：
- 权益卡片：2 列布局
- 预览卡片：2 列布局
- 评价卡片：2 列布局

手机端（< 768px）：
- 权益卡片：单列布局
- 预览卡片：单列布局
- 评价卡片：单列布局
```
