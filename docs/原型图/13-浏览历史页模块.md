## 🎨 二、组件使用详情

### 组件 1：历史卡片

**使用组件：** @component-card + @component-badge

**布局结构：**
```
┌───────────────────┐
│  [封面图]         │
│  AI 写作助手       │
│  🚀 项目          │
│  ─────────────────│
│  10:30 浏览        │
│  ─────────────────│
│  [AI 工具] [写作] │
│  ─────────────────│
│        [🗑️ 删除]  │
└───────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 历史卡片 | 悬停 | 上移 4px + 阴影加深 | 无 |
| 封面图 | 点击 | 无 | 跳转对应内容页 |
| 标题 | 点击 | 无 | 跳转对应内容页 |
| 删除 | 点击 | 弹出确认框 | DELETE /api/history/{id} |

---

### 组件 2：清空历史

**使用组件：** @component-button（次按钮）

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [清空历史]                             │
│  ─────────────────────────────────────  │
│  点击弹出确认框：                       │
│  "确定要清空所有浏览历史吗？            │
│   此操作不可恢复"                       │
│  ─────────────────────────────────────  │
│  [取消]  [确认清空]                     │
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 清空历史 | 点击 | 弹出确认框 | 无 |
| 取消 | 点击 | 关闭弹窗 | 无 |
| 确认清空 | 点击 | 显示"清空中..." | DELETE /api/history/clear |
| 清空成功 | - | 显示"已清空" | 刷新列表为空 |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 筛选器 | 类型下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 日期下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 搜索框 | 输入 | 实时建议 | GET /api/history/suggest |
| 历史卡片 | 卡片 | 悬停 | 上移 4px | 无 |
| 历史卡片 | 标题 | 点击 | 无 | 跳转对应内容页 |
| 历史卡片 | 删除 | 点击 | 弹出确认框 | DELETE /api/history/{id} |
| 清空历史 | 按钮 | 点击 | 弹出确认框 | DELETE /api/history/clear |
| 分页器 | 页码 | 点击 | 高亮当前页 | 加载对应页数据 |

## 🔌 四、API 接口

### 浏览历史列表接口

**请求：**
```http
GET /api/history?page=1&per_page=9&type=all&date=today
```

**响应：**
```json
{
  "success": true,
  "data": {
    "histories": [
      {
        "id": 1,
        "viewable_type": "App\\Models\\Project",
        "viewable_id": 1,
        "title": "AI 写作助手",
        "cover_image": "/covers/1.jpg",
        "category": "project",
        "tags": ["AI 工具", "写作"],
        "viewed_at": "2026-04-03 10:30:00"
      }
    ],
    "pagination": {
      "total": 128,
      "per_page": 9,
      "current_page": 1,
      "last_page": 15
    }
  }
}
```

### 清空历史接口

**请求：**
```http
DELETE /api/history/clear
```

**响应：**
```json
{
  "success": true,
  "message": "已清空所有浏览历史"
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 历史卡片：3 列布局
- 筛选器：横向布局
- 分页器：完整显示

平板端（768px - 1023px）：
- 历史卡片：2 列布局
- 筛选器：换行显示
- 分页器：简化显示

手机端（< 768px）：
- 历史卡片：单列布局
- 筛选器：折叠显示
- 分页器：仅显示上一页/下一页
```
