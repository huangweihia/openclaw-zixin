## 🎨 二、组件使用详情

### 组件 1：邮箱输入框 + 获取验证码按钮

**使用组件：** @component-input + @component-button（次按钮）

**布局结构：**
```
┌─────────────────────────────────────────┐
│  邮箱地址 *                             │
│  ┌─────────────────┐ ┌──────────────┐ │
│  │ your@email.com  │ │[获取验证码]  │ │
│  └─────────────────┘ └──────────────┘ │
└─────────────────────────────────────────┘
```

**组件属性：**
- 邮箱输入框：宽度 70%，高度 56px
- 验证码按钮：宽度 28%，高度 56px
- 间距：2%

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 邮箱输入框 | 输入 | 实时验证格式 | 格式错误显示红色边框 |
| 邮箱输入框 | 失焦 | 验证邮箱格式 | 错误显示提示 |
| 获取验证码 | 点击 | 验证邮箱格式 | POST /register/send-code |
| 获取验证码（成功） | - | 显示"已发送" + 倒计时 60s | 按钮禁用 |
| 获取验证码（失败） | - | 显示错误提示 | 按钮恢复可用 |
| 获取验证码（倒计时） | - | 显示"60s 后重发" | 按钮禁用 |

**验证规则：**
- 邮箱必填
- 邮箱格式验证
- 邮箱唯一性检查（已注册提示"该邮箱已注册"）

---

### 组件 2：密码强度指示器

**使用组件：** @component-password-strength

**布局结构：**
```
┌─────────────────────────────────────────┐
│  密码 *                    [弱|中|强]   │
│  ┌─────────────────────────────────┐   │
│  │ ••••••••                        │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**组件属性：**
- 强度指示器：右侧显示
- 弱：红色背景
- 中：橙色背景
- 强：绿色背景

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 密码输入框 | 输入 | 实时计算强度 | 更新指示器颜色 |
| 强度指示器 | - | 根据密码长度变化 | 无 |

**强度规则：**
- 弱：< 6 字符
- 中：6-10 字符
- 强：> 10 字符 + 包含大小写/数字/特殊字符

## 🔌 四、API 接口

### 1. 发送验证码接口

**请求：**
```http
POST /api/register/send-code
Content-Type: application/json

{
  "email": "user@example.com"
}
```

**响应（成功）：**
```json
{
  "success": true,
  "message": "验证码已发送，有效期 5 分钟",
  "data": {
    "expires_in": 300
  }
}
```

**响应（失败）：**
```json
{
  "success": false,
  "message": "该邮箱已注册"
}
```

---

### 2. 注册接口

**请求：**
```http
POST /api/register
Content-Type: application/json

{
  "email": "user@example.com",
  "verification_code": "123456",
  "password": "password123",
  "password_confirmation": "password123",
  "name": "张三"
}
```

**响应（成功）：**
```json
{
  "success": true,
  "data": {
    "user": {
      "id": 1,
      "name": "张三",
      "email": "user@example.com",
      "vip_level": "free"
    },
    "token": "eyJ0eXAiOiJKV1QiLCJhbGc..."
  },
  "message": "注册成功"
}
```

**响应（失败）：**
```json
{
  "success": false,
  "message": "注册失败",
  "errors": {
    "verification_code": ["验证码错误或已过期"],
    "password": ["密码长度不足"]
  }
}
```
