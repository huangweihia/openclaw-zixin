## 🎨 二、组件使用详情

### 组件 1：项目卡片

**使用组件：** @component-card + @component-badge

**布局结构：**
```
┌───────────────────┐
│  [封面图]         │
│  AI 写作助手       │
│  ─────────────────│
│  💰 月收入 5000+  │
│  ─────────────────│
│  启动：0 元        │
│  难度：⭐⭐         │
│  ─────────────────│
│  [AI 工具] [写作] │
│  ⭐ 收藏           │
└───────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 项目卡片 | 悬停 | 上移 4px + 阴影加深 | 无 |
| 封面图 | 点击 | 无 | 跳转项目详情 |
| 标题 | 点击 | 无 | 跳转项目详情 |
| 收藏 | 点击 | 变色 | POST /api/projects/{id}/favorite |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |
| 导航栏 | 项目 | 点击 | 文字变紫色 | 刷新当前页 |
| 筛选器 | 分类下拉 | 点击 | 展开列表 | 无 |
| 筛选器 | 搜索框 | 输入 | 实时建议 | GET /api/projects/suggest |
| 项目卡片 | 卡片 | 悬停 | 上移 4px | 无 |
| 项目卡片 | 标题 | 点击 | 无 | 跳转项目详情 |
| 项目卡片 | 收藏 | 点击 | 变色 | POST /api/projects/{id}/favorite |
| 分页器 | 页码 | 点击 | 高亮当前页 | 加载对应页数据 |

## 🔌 四、API 接口

### 项目列表接口

**请求：**
```http
GET /api/projects?page=1&per_page=9&category=all&sort=latest
```

**响应：**
```json
{
  "success": true,
  "data": {
    "projects": [
      {
        "id": 1,
        "title": "AI 写作助手",
        "cover_image": "/covers/1.jpg",
        "monthly_income": "5000+",
        "startup_cost": 0,
        "difficulty": 2,
        "tags": ["AI 工具", "写作"],
        "is_favorite": false
      }
    ],
    "pagination": {
      "total": 100,
      "per_page": 9,
      "current_page": 1,
      "last_page": 12
    }
  }
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 项目卡片：3 列布局
- 筛选器：横向布局
- 分页器：完整显示

平板端（768px - 1023px）：
- 项目卡片：2 列布局
- 筛选器：换行显示
- 分页器：简化显示

手机端（< 768px）：
- 项目卡片：单列布局
- 筛选器：折叠显示
- 分页器：仅显示上一页/下一页
```
