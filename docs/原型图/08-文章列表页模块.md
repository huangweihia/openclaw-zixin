## 🎨 二、组件使用详情

### 组件 1：筛选器

**使用组件：** @component-select + @component-search-box

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [全部分类▼]  [全部标签▼]  [🔍 搜索...] │
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 分类下拉 | 点击 | 展开分类列表 | 无 |
| 标签下拉 | 点击 | 展开标签列表 | 无 |
| 搜索框 | 点击 | 聚焦，显示光标 | 无 |
| 搜索框 | 输入 | 实时搜索建议 | GET /api/articles/suggest |
| 搜索框 | 回车 | 执行搜索 | GET /api/articles?q=xxx |

---

### 组件 2：文章卡片

**使用组件：** @component-card + @component-badge

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [封面图]  《AI 副业变现的 5 个方向》     │
│  ─────────────────────────────────────  │
│  💰 副业案例 | 📅 2026-04-03 | 👁️ 1,234│
│  ─────────────────────────────────────  │
│  摘要：本文介绍了 5 个 AI 副业变现方向...  │
│  ─────────────────────────────────────  │
│  [AI 副业] [变现] [案例]          ⭐ 收藏│
└─────────────────────────────────────────┘
```

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 文章卡片 | 悬停 | 上移 4px + 阴影加深 | 无 |
| 封面图 | 点击 | 无 | 跳转文章详情 |
| 标题 | 点击 | 无 | 跳转文章详情 |
| 标签 | 点击 | 无 | 筛选该标签 |
| 收藏 | 点击 | 变色（已收藏） | POST /api/articles/{id}/favorite |

## 📋 三、交互说明表

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| 导航栏 | Logo（🦀） | 点击 | 无 | 刷新首页 |

| 文章卡片 | 收藏 | 点击 | 变色 | POST /api/articles/{id}/favorite |
| 分页器 | 页码 | 点击 | 高亮当前页 | 加载对应页数据 |

## 🔌 四、API 接口

### 文章列表接口

**请求：**
```http
GET /api/articles?page=1&per_page=10&category=all&tag=all&sort=latest
```

**响应：**
```json
{
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 文章卡片：单列
- 筛选器：横向布局
- 分页器：完整显示

平板端（768px - 1023px）：
- 文章卡片：单列
- 筛选器：换行显示
- 分页器：简化显示

手机端（< 768px）：
- 文章卡片：单列
- 筛选器：折叠显示
- 分页器：仅显示上一页/下一页
```
