
## 🎨 二、组件使用详情

### 组件 1：Hero Section

**使用组件：** @component-button

**布局结构：**
```
┌─────────────────────────────────────────┐
│          OpenClaw 智信（H1）             │
│  OpenClaw + AI 智能体...（正文大）       │
│                                         │
│  [免费试用 7 天]  [看看有什么]           │
│  (@component-button 主按钮)             │
└─────────────────────────────────────────┘
```

**组件属性：**
- 背景：渐变背景 (#667eea → #764ba2)
- 高度：600px
- 文字：白色

---

### 组件 2：数据展示区

**使用组件：** @component-font, @component-color

**布局结构：**
```
┌─────────────────────────────────────────┐
│  1,234+    50+     20+     500 万 +     │
│  注册用户  副业案例  工具地图  累计变现  │
│  (H2)      (H2)    (H2)    (H2)        │
└─────────────────────────────────────────┘
```

**组件属性：**
- 数字：48px/Bold/渐变文字
- 说明：16px/Regular/灰色

---

### 组件 3：VIP 内容预览区

**使用组件：** @component-card

**布局结构：**
```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│  副业案例     │  │  AI 工具      │  │  运营 SOP     │
│  《小红书...》 │  │  《Midjourney》│  │  《小红书...》 │
│  启动成本：0 元│  │  5 大变现场景  │  │  30 天万粉    │
│  [开通 VIP]   │  │  [开通 VIP]   │  │  [开通 VIP]   │
│  (@component-button)              │
└──────────────┘  └──────────────┘  └──────────────┘
```

**组件属性：**
- 卡片：@component-card
- 按钮：@component-button 次按钮
- 锁定标识：🔒 (@component-icon)

---

### 组件 4：价格方案区

**使用组件：** @component-card, @component-button

**布局结构：**
```
┌────────┐  ┌────────────┐  ┌────────┐
│ 🆓 免费 │  │ 💎 VIP     │  │ 👑 SVIP│
│ 0 元    │  │ 199 元/年   │  │ 999 元/年│
│ 永久免费│  │ 限时优惠   │  │ 仅限 50 人│
│ [免费注册]│  │ [立即开通] │  │ [申请 SVIP]│
│ (@component-button)                │
└────────┘  └────────────┘  └────────┘
```

**组件属性：**
- 卡片：@component-card 价格卡片
- 按钮：@component-button 主按钮/次按钮
- 推荐标识：🔥 (@component-icon)

## 📋 三、交互说明

| 区域 | 元素 | 点击/悬停 | 反应 | 动作 |
|------|------|----------|------|------|
| Hero Section | 免费试用 7 天 | 点击 | 无 | 跳转注册页 |
| Hero Section | 看看有什么 | 点击 | 平滑滚动 | 滚动到 VIP 预览区 |
| 数据展示区 | - | - | - | 纯展示 |
| VIP 预览区 | 开通 VIP | 点击 | 无 | 跳转价格页 |
| 价格方案区 | VIP 方案 | 悬停 | 卡片上移 + 阴影加深 | 无 |
| 价格方案区 | VIP 方案 | 点击 | 无 | 跳转支付页 |
