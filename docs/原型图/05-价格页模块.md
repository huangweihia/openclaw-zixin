## 🎨 二、组件使用详情

### 组件 1：价格卡片

**使用组件：** @component-pricing-card

**布局结构：**
```
┌─────────────────────────────────────────┐
│  🔥 最受欢迎（仅 VIP 卡片）              │

│  ─────────────────────────────────────  │
│  ✅ 免费版全部权益                      │

│  ⏰ 限时优惠，还剩 23 个名额              │

```

**组件属性：**
- 宽度：320px
- 高度：600px
- 背景：白色
- 圆角：16px
- 阴影：shadow-2xl
- 边框：VIP 卡片 4px 紫色边框
- 悬停：上移 8px + shadow-2xl

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 价格卡片 | 悬停 | 上移 8px + 阴影加深 | 无 |
| 立即开通 VIP | 点击 | 无 | 跳转支付页（?plan=vip） |
| 申请 SVIP | 点击 | 无 | 跳转申请页 |
| 免费注册 | 点击 | 无 | 跳转注册页 |

## 📱 八、响应式规则

### 断点定义

```
sm: 640px   （手机横屏）
md: 768px   （平板）
lg: 1024px  （桌面）
xl: 1280px  （大桌面）
2xl: 1536px （超大桌面）
```

### 响应式规则

```
桌面端（>= 1024px）：
- 价格卡片：3 列布局
- VIP 卡片：放大 1.05 倍 + 紫色边框

平板端（768px - 1023px）：
- 价格卡片：2 列布局
- VIP 卡片：居中显示

手机端（< 768px）：
- 价格卡片：单列布局
- VIP 卡片：正常大小
```
