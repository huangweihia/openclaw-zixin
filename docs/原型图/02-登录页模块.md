## 🎨 二、组件使用详情

### 组件 1：邮箱输入框

**使用组件：** @component-input

**布局结构：**
```
┌─────────────────────────────────────────┐
│  邮箱地址 *                             │
│  ┌─────────────────────────────────┐   │
│  │ your@email.com                  │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**组件属性：**
- 宽度：100%
- 高度：56px
- 圆角：8px
- 边框：1px solid #e5e7eb
- 聚焦：紫色边框 + 阴影

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 输入框 | 点击 | 聚焦状态（紫色边框） | 显示光标 |
| 输入框 | 输入 | 实时验证格式 | 格式错误显示红色边框 |
| 输入框 | 失焦 | 验证邮箱格式 | 错误显示"请输入正确的邮箱" |

**验证规则：**
- 必填
- 邮箱格式（正则：`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`）
- 邮箱需已注册（未注册提示"该邮箱未注册，请先注册"）

## 🔌 七、API 接口

### 登录接口

**请求：**
```http
POST /api/login
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "password123",
  "remember": true
}
```

**响应（成功）：**
```json
{
  "success": true,
  "data": {
    "user": {
      "id": 1,
      "name": "张三",
      "email": "user@example.com",
      "avatar": "/avatars/1.png",
      "vip_level": "vip"
    },
    "token": "eyJ0eXAiOiJKV1QiLCJhbGc..."
  },
  "message": "登录成功"
}
```

**响应（失败）：**
```json
{
  "success": false,
  "message": "邮箱或密码错误",
  "errors": {
    "email": ["该邮箱未注册"]
  }
}
```

## 📱 八、响应式规则

### 断点定义

```
sm: 640px   （手机横屏）
md: 768px   （平板）
lg: 1024px  （桌面）
xl: 1280px  （大桌面）
2xl: 1536px （超大桌面）
```

### 响应式规则

```
桌面端（>= 1024px）：
- 表单卡片宽度：480px
- 居中布局
- 完整导航栏

平板端（768px - 1023px）：
- 表单卡片宽度：400px
- 居中布局
- 汉堡菜单

手机端（< 768px）：
- 表单卡片宽度：100% - 32px
- 单列布局
- 底部导航（可选）
```
