## 🎨 二、组件使用详情

### 组件 1：发布列表

**使用组件：** @component-publish-list

**布局结构：**
```
┌─────────────────────────────────────────┐
│  [封面图]  《我的副业案例分享》          │
│  ─────────────────────────────────────  │
│  📝 副业案例 | 🟡 审核中 | 📅 2026-04-03│
│  ─────────────────────────────────────  │
│  👁️ 0  |  👍 0  |  💬 0                 │
│  ─────────────────────────────────────  │
│          [✏️ 编辑] [🗑️ 删除] [👁️ 查看]  │
└─────────────────────────────────────────┘
```

**状态标识：**

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 审核中 | 🟡 | 橙色 | 等待管理员审核 |
| 已通过 | 🟢 | 绿色 | 审核通过，已公开 |
| 已拒绝 | 🔴 | 红色 | 审核未通过，需修改 |

**交互说明：**

| 元素 | 点击/悬停 | 反应 | 动作 |
|------|----------|------|------|
| 发布卡片 | 悬停 | 上移 4px + 阴影加深 | 无 |
| 封面图 | 点击 | 无 | 跳转内容详情页 |
| 标题 | 点击 | 无 | 跳转内容详情页 |
| 编辑 | 点击 | 无 | 跳转编辑页 |
| 删除 | 点击 | 弹出确认框 | DELETE /api/publishes/{id} |
| 查看 | 点击 | 无 | 跳转内容详情页 |

**请求：**
```http
GET /api/publishes?page=1&per_page=10&status=all
```

**响应：**
```json

  "data": {

        "id": 1,
```

### 删除发布接口

**请求：**
```http
DELETE /api/publishes/{id}
```

**响应：**
```json
{
  "success": true,
  "message": "已删除发布"
}
```

## 📱 五、响应式规则

```
桌面端（>= 1024px）：
- 发布卡片：单列
- 筛选器：横向布局
- 分页器：完整显示

平板端（768px - 1023px）：
- 发布卡片：单列
- 筛选器：换行显示
- 分页器：简化显示

手机端（< 768px）：
- 发布卡片：单列
- 筛选器：折叠显示
- 分页器：仅显示上一页/下一页
```
