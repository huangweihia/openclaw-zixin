# 后台管理 ↔ 前台功能对应说明

以下为常见后台模块与前台展示/能力的对应关系，便于运营与开发对照。路径以管理端 Vue 路由或 Laravel 菜单为准。

## 系统与站点（Settings）

| 后台配置项 | 前台作用 |
|------------|----------|
| 站点名称、标语、简介 | 顶栏品牌文案、首页 Hero、页脚描述等（`SiteViewComposer` → `$ocSite`） |
| **站点 Logo 地址**（`site_logo_url`） | 顶栏小图标、页脚图标、首页 Hero 左侧图标；留空则用默认 `favicon.svg`；支持完整 URL 或 `/storage/...` 相对路径 |
| 联系邮箱、微信、页脚 HTML | 页脚「联系我们」与附加说明 |
| 注册赠送会员、积分 | 注册成功后的角色与积分流水（`RegisterController`） |
| 价格营销（截止时间与名额、副标题） | 首页与 VIP 相关价格区的倒计时与营销文案（`PricingConfig` / `site_settings`） |
| 邮件批处理时间窗 | 定时任务发送邮件时段控制（`Kernel` / 相关命令读取） |

## 公告管理

| 配置 | 前台 |
|------|------|
| 已发布 + 未过期 | 才可能展示 |
| 展示位置 = 顶部/底部 + **关闭浮动** | `partials/announcement-marquee`（顶/底各一条轨道，多条公告在同一轨道内滚动） |
| 浮动展示 = 是 | `partials/announcement-float`（角标/侧栏卡片，位置由 `display_position` 的 left/right/bottom 等决定） |

## 广告位

| 后台 | 前台 |
|------|------|
| 管理端「广告位」`/ad-slots` | 前台按广告位的 `position/width/height` 与兜底素材（图片/链接/文案）渲染 |
| 启用状态 `is_active` | 全局同一时间仅允许 1 个广告位启用，启用新广告位会自动停用其它广告位 |

## 首页评价（Site Testimonials）

| 后台 | 前台 |
|------|------|
| 管理端「首页评价」`/site-testimonials` | 首页「用户评价」区块；表 `site_testimonials`，已发布且排序；无数据时 `HomeController` 可写入演示数据 |

## 用户、评论、举报、邮件记录

| 后台页面 | 说明 |
|----------|------|
| 用户管理 | 前台账号体系、角色、企微授权状态等 |
| 评论管理 | 全站评论可见性（隐藏/删除）；列表分页使用 `AdminPagination` |
| 评论举报 | 举报单处理状态 |
| 邮件发送记录 | 系统发信队列/结果审计；分页使用 `AdminPagination` |

## 换肤与主题色

前台通过 `data-skin` 与 `public/css/skins.css` 中的 `--primary`、`--gradient-primary` 等变量换肤。首页 VIP 区块、价格方案 CTA 等使用 `.oc-cta-primary`、`.oc-hero-cta-solid` 等类名，**随当前皮肤主题色变化**，避免写死紫色/绿色。

## 文档维护说明

新增后台菜单时，建议在本文件追加一行「后台路径 → 前台路由/视图/API」，保持运营可查。
