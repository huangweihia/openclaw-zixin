## 📐 五、数据库表设计

### view_histories 表

```sql
CREATE TABLE view_histories (
    id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT COMMENT '历史 ID',
    user_id BIGINT UNSIGNED NOT NULL COMMENT '用户 ID',
    viewable_type VARCHAR(255) NOT NULL COMMENT '内容类型',
    viewable_id BIGINT UNSIGNED NOT NULL COMMENT '内容 ID',
    viewed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '浏览时间',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    INDEX idx_user_id (user_id),
    INDEX idx_viewable (viewable_type, viewable_id),
    INDEX idx_viewed_at (viewed_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='浏览历史表';
```

**字段说明：**

| 字段 | 类型 | 必填 | 默认值 | 说明 | 示例 |
|------|------|------|--------|------|------|
| id | BIGINT | ✅ | 自增 | 历史 ID | 1 |
| user_id | BIGINT | ✅ | - | 用户 ID | 1 |
| viewable_type | VARCHAR | ✅ | - | 内容类型 | App\Models\Article |
| viewable_id | BIGINT | ✅ | - | 内容 ID | 123 |
| viewed_at | TIMESTAMP | ✅ | CURRENT | 浏览时间 | 2026-04-03 10:00:00 |

**业务规则：**
1. 同一内容 24 小时内不重复记录
2. 删除用户时自动删除历史（ON DELETE CASCADE）
3. 隐私模式开启时不记录
4. 仅保留最近 180 天的历史记录（定时清理）
