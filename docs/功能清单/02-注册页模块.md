## 功能 2：获取验证码按钮

| 项目 | 内容 |
|------|------|
| **功能描述** | 点击发送邮箱验证码 |
| **页面位置** | 邮箱输入框右侧 |
| **涉及数据库** | 缓存：email_code_{email} |
| **交互详情** | <table><tr><th>元素</th><th>点击/悬停</th><th>反应</th><th>跳转/动作</th></tr><tr><td>**获取验证码**</td><td>点击</td><td>按钮变灰 + 倒计时</td><td>POST /register/send-code</td></tr><tr><td>**获取验证码（成功）**</td><td>-</td><td>显示"已发送"</td><td>按钮禁用 60 秒</td></tr><tr><td>**获取验证码（失败）**</td><td>-</td><td>显示错误提示</td><td>按钮恢复可用</td></tr></table> |
| **API 调用** | POST /register/send-code<br>Body: { email: "user@example.com" } |
| **数据逻辑** | 1. 验证邮箱格式<br>2. 查询 users.email 检查是否已注册<br>3. 检查发送频率（60 秒限制）<br>4. 生成 6 位验证码<br>5. **先** 调用 SMTP 发信，**成功后再** 写入缓存（5 分钟）与 60 秒频控；失败返回 **422**，前端展示错误（不误报「已发送」）<br>6. **运维：** 日志栈兜底见 `config/logging.php`；QQ 邮箱需授权码且 **`MAIL_FROM_ADDRESS` 宜与发信账号一致** |
| **开发状态** | ✅ 已完成 |
| **测试状态** | ⬜ 未测试 |
