| 项目 | 内容 |
|------|------|
| **功能描述** | 发表对文章/项目的评论 |
| **页面位置** | 文章/项目详情页评论区 |
| **涉及数据库** | 表：comments<br>字段：comments.user_id, comments.commentable_type, comments.commentable_id, comments.content, comments.parent_id |
| **交互详情** | <table><tr><th>元素</th><th>点击/悬停</th><th>反应</th><th>跳转/动作</th></tr><tr><td>**评论输入框**</td><td>点击</td><td>显示输入光标</td><td>无</td></tr><tr><td>**评论输入框**</td><td>输入</td><td>实时显示输入内容</td><td>无</td></tr><tr><td>**发表评论**</td><td>点击（未登录）</td><td>显示"请先登录"弹窗</td><td>跳转登录页</td></tr><tr><td>**发表评论**</td><td>点击（已登录）</td><td>显示"发表中..."</td><td>POST /articles/{id}/comments</td></tr><tr><td>**发表成功**</td><td>自动</td><td>显示"发表成功"</td><td>刷新评论列表</td></tr></table> |

|------|------|
| **功能描述** | 举报不良评论 |
| **页面位置** | 评论下方 |

| **交互详情** | <table><tr><th>元素</th><th>点击/悬停</th><th>反应</th><th>跳转/动作</th></tr><tr><td>**举报**</td><td>点击</td><td>显示举报弹窗（`oc-modal`）</td><td>无</td></tr><tr><td>**举报原因**</td><td>选择</td><td>下拉选中</td><td>无</td></tr><tr><td>**补充说明**</td><td>选填</td><td>多行文本，≤500 字</td><td>写入 `comment_reports.description`</td></tr><tr><td>**提交举报**</td><td>点击</td><td>关闭弹窗；**右上角 Toast** 成功提示（与 `00` **功能 16** 规格：3s、悬停暂停、点按关闭）</td><td>`POST` 见下</td></tr></table> |
