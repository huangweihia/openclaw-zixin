## 功能 1：皮肤切换（后台配置）

| 项目 | 内容 |
|------|------|
| **功能描述** | 切换网站皮肤（从后台 `skin_configs` 表动态读取，不再写死） |
| **页面位置** | 导航栏右侧 / 个人中心 - 设置页 |
| **涉及数据库** | 表：`skin_configs`（后台配置）+ `user_theme_preferences`（用户偏好）<br>字段：`skin_configs.code`, `skin_configs.name`, `skin_configs.css_variables`, `skin_configs.is_enabled` |
| **交互详情** | <table><tr><th>元素</th><th>点击/悬停</th><th>反应</th><th>跳转/动作</th></tr><tr><td>**皮肤选择器**</td><td>点击</td><td>展开皮肤列表</td><td>GET /api/skins</td></tr><tr><td>**皮肤选项**</td><td>悬停</td><td>预览边框高亮</td><td>无</td></tr><tr><td>**皮肤选项**</td><td>点击</td><td>立即切换皮肤</td><td>PUT /user/skin + 应用 CSS 变量</td></tr><tr><td>**当前皮肤**</td><td>-</td><td>显示选中标记</td><td>无</td></tr></table> |
| **API 调用** | `GET /api/skins` - 获取启用的皮肤列表<br>`PUT /user/skin` - 保存用户皮肤偏好<br>Body: `{ skin_code: "vip-gold" }` |
| **数据逻辑** | 1. 页面加载时调用 `GET /api/skins` 获取启用的皮肤列表<br>2. 用户选择后调用 `PUT /user/skin` 保存偏好<br>3. 动态应用 `skin_configs.css_variables` JSON 中的 CSS 变量<br>4. 皮肤切换立即生效（无需刷新） |
| **皮肤配置（后台）** | 后台管理 `skin_configs` 表，字段：<br>- `name`: 皮肤名称（如：鎏金 VIP、深空蓝）<br>- `code`: 皮肤代码（如：vip-gold、deep-space-blue）<br>- `type`: 类型（free / vip / svip）<br>- `is_enabled`: 是否启用<br>- `css_variables`: JSON 对象（CSS 变量定义）<br>- `sort_order`: 排序 |
| **开发状态** | ✅ 已完成 |
| **测试状态** | ✅ 测试通过 |
