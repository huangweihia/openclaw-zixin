## 📋 原型图对照（`docs/原型图/11-项目详情页模块.md`）

> 核对代码：`_laravel_temp/resources/views/projects/show.blade.php`、`ProjectController@show`（2026-04-05）

| 原型要素 | 当前实现 | 差异说明 |
|---------|---------|---------|
| 顶栏 Hero 区 | 标题 + 分类链 + Star/Fork/语言 + 外链 | **无**大图封面、**无**原型中的「💰 月收入 | 启动 | 难度」一行摘要、**无**分享按钮 |
| 项目详情 / 变现 | `oc-surface` 多段文案 | **无**「操作步骤」独立章节；变现分析非 stat-card 三列布局 |
| 评论 | 表单 + `partials.comment-row` + 分页 | 与原型评论区结构接近；**无**评论用户 VIP 角标链到个人主页 |
| 侧栏相关项目 | `sticky` 列表 | 有；悬停样式为 `oc-related-row`，非独立卡片上移动效文案 |
