## 📋 原型图对照（`docs/原型图/10-项目列表页模块.md`）

> 核对代码：`_laravel_temp/resources/views/projects/index.blade.php`、`ProjectController@index`（2026-04-05）

| 原型要素 | 当前实现 | 差异说明 |
|---------|---------|---------|
| 顶栏 / 页脚 | `layouts.site` + `x-site-navbar` | 与全站一致，非独立 `@component-top-nav` 标签 |
| 筛选区 | 分类 Pill、排序（Star/最新）、搜索框 + 提交 | 原型中的「全部标签▼」「排序▼」下拉：现为 Pill 链接；**额外支持** `language` 查询参数筛主语言 |
| 搜索防抖 | 输入 500ms 后 `requestSubmit` | 与清单一致 |
| 项目卡片 | `oc-article-card` 网格 + 占位图 | 原型中的封面图、💰 月收入、启动成本、难度星级、卡片内「收藏」：**当前卡片为简化版**（标题、摘要、语言、Star/Fork） |
| 分页 | Laravel 分页链接 | 与原型分页区一致（样式为默认 / utility） |
